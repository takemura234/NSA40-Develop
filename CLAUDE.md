# CLAUDE.md

## プロジェクト概要

開発中の柱2柱門型リフト **NSA40** に関する検証リポジトリ。
耐久試験・強度検証・安全性検討などの結果を関係メンバー間で共有・議論し、
開発判断の見える化を進めることを目的としている。

**注意：機密情報を含む可能性があるため、社外共有には注意すること。**

## ファイル構成

```
NSA40-Develop/
├── Doc/
│   ├── TestData/   ← 試験データ・検証結果（Markdown + PDF）
│   ├── 議事録/     ← 会議記録（Format.mdがテンプレート）
│   ├── 仕様書/     ← 設計・仕様関連資料
│   └── 参考資料/   ← 法規・リスク確認等の参考文献
├── image/          ← 試験・現場写真
└── README.md
```

## 主な検証テーマ

- 耐久試験結果
- 強度検証
- 安全性検討
- 法規・リスク確認
- 設計レビュー
- 課題管理
- 試験データ共有

## 運用ルール

- 検証データは `Doc/TestData/` に Markdown（または PDF）で追加する
- 議事録は `Doc/議事録/Format.md` をテンプレートとして使用する
- 写真は容量を抑えて `image/` に追加する
- 機密性の高い内容は Issues や Pull Request のコメントに書かない

## コミットメッセージルール

```
docs: NSA40チェーン破断検証の結果を追加
test: 断流弁作動確認データを更新
fix: 議事録2026-05-21の誤記を修正
```

何をしたか一目でわかるメッセージを書くこと。
